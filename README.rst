epaper telegram
===================

communicate between 2 rasperry pi with a touch epaper display (waveshare). Draw something nice, and it will be sent to the other device!


Installation
============

run install.sh
* clone
* env
* pip
* create env
* activate
* pip install
* symbolic link in bin
* crontab at startup
(method to pair a device?)
(cloud, link? pass?)

hardware requirements

* waveshare
